# Work

Active work. Changes constantly.

## Structure

```
work/
├── campaigns/    # time-bound efforts (launches, pushes)
├── projects/     # larger initiatives spanning weeks
├── research/     # market analysis, discovery, competitive deep dives
└── content/      # blog posts, pitches, copy, changelogs
```

## Where does it go?

| Type | Folder | Example |
|------|--------|---------|
| A launch push | `campaigns/` | `campaigns/q1-growth/` |
| A multi-week initiative | `projects/` | `projects/website-redesign/` |
| Market research | `research/` | `research/2026-03-competitor-analysis.md` |
| A blog post or pitch | `content/` | `content/blog-why-context-matters.md` |

## Conventions

- **Campaigns** get their own folder with a README
- **Projects** get their own folder with a README
- **Research** files are named `YYYY-MM-topic.md`
- **Content** files are named by type and topic
