# Market

What we know about our market, customers, and competition.

## Files

| File | Purpose |
|------|---------|
| `icp.md` | Ideal customer profile — who we're building for |
| `competitors/` | One file per competitor with structured analysis |

## How to use

- Read `icp.md` before any sales, outreach, or positioning work
- Check `competitors/` before competitive conversations or positioning updates
- Update these files when you learn something new — don't let them go stale
