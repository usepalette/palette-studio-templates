# Competitors

One file per competitor. Structured analysis for quick reference.

## File format

Name each file by the competitor's name in lowercase: `competitor-name.md`

## Template

```markdown
# [Competitor Name]

## Overview
- **What they do:** [One sentence]
- **Founded:** [Year]
- **Size:** [Employees / stage]
- **Funding:** [Amount / stage]
- **Website:** [URL]

## Positioning
[How they describe themselves. Use their own words.]

## Target audience
[Who they sell to — size, industry, role]

## Product
[What the product actually does. Be specific.]

## Strengths
- [What they do well]

## Weaknesses
- [Where they fall short]

## How we're different
[Direct comparison — why would someone choose us over them?]

## Notes
[Anything else useful — recent moves, pricing, market perception]
```
